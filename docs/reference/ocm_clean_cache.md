## ocm clean cache &mdash; Cleanup Oci Blob Cache

### Synopsis

```
ocm clean cache [<options>]
```

### Options

```
  -h, --help   help for cache
```

### Description


Cleanup all blobs stored in oci blob cache (if given).
	

### Examples

```

$ ocm clean cache

```
