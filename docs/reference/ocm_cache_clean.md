## ocm cache clean &mdash; Cleanup Oci Blob Cache

### Synopsis

```
ocm cache clean [<options>]
```

### Options

```
  -h, --help   help for clean
```

### Description


Cleanup all blobs stored in oci blob cache (if given).
	

### Examples

```

$ ocm clean cache

```
