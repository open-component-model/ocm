## ocm cache info &mdash; Show OCI Blob Cache Information

### Synopsis

```
ocm cache info [<options>]
```

### Options

```
  -h, --help   help for info
```

### Description


Show details about the OCI blob cache (if given).
	

### Examples

```

$ ocm cache info

```
